Começo rápido

Assegure-se de ter Ambiente de verificação Depois disso, aqui está um caso mais simples: use QScan Digitalize o código para fazer login na ferramenta de desenvolvedor WeChat, use a linha de comando para chamar o serviço de digitalização de código, instale Ferramentas de desenvolvedor WeChat,

Preencha a configuração

editar ~/.qscanrc Arquivo, preenchaConfiguração 

vi ~/.qscanrc

{
    "wx-default": { // key O valor é o nome modal
        "udid": "HICMHMZTTW8DFI59", // ID do dispositivo Android, visualizar por meio do comando de serviço adb
        "port": "4723", // Número da porta, por exemplo, definido aqui como 4723
        "opts": {
            "user": "xxx", // Nome de usuário do WeChat
            "pass": "xxx" // Senha do WeChat
        },
        "checkApp": true // Verifique se deseja instalar o WeChatAPP E sua versão 
    }
}

comece Linus serviço

Inicie o serviço Appium de acordo com o número da porta e o id do dispositivo configurado no arquivo .qscanrc agora mesmo 

appium -p 4723 -U HICMHMZTTW8DFI59

Verifique o ambiente operacional

Depois de instalar o QScan globalmente, execute qscan doctor Verifique a configuração acima para garantir que todas as saídas sejam ✔ SUCCESS

Abra as ferramentas de desenvolvedor do WeChat

Abra a página de login do WeChat Developer Tools, ajuste o local e espere que o código seja verificado

Executar serviço de leitura de código

Execute os seguintes códigos, que são o serviço de código de varredura WeChat fornecido pelo QScan para o modelo especificado, digite O tipo é ide-login-scan (WeChat Developer Tools) 

qscan scan -m wx-default -t ide-login-scan

Você pode ver o login automático no telefone celular e o código de varredura bem-sucedido, e você pode ver o registro de saída c orrespondente no terminal.

Um dos serviços de digitalização de código mais simples está instalado e funcionando. Além de usar a linha de comando, você também pode usar o node O módulo chama QScan, porque QScan O código principal do é relativamente puro, então você pode usar sua imaginação para aplicá-lo aos seus cenários de negócios.
