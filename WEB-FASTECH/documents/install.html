instalação

Para usar o Linux, você precisa preparar alguns equipamentos e instalar algumas ferramentas com antecedência

Ferramenta de instalação

QScan Confie nos seguintes equipamentos e ferramentas：

equipamento versão

Android equipamento >=v4.4
ferramenta versão
Node >=v8.0
Appium A última versão

1. instalação Node.js

Programa一: De Node.js Baixe o pacote de instalação do site oficial

Programa二: usar Node Version Manager(NVM) instalação Node.js，Você pode instalá-lo por NVM :

cURL:

$ curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.33.2/install.sh | bash

Wget:

$ wget -qO- https://raw.githubusercontent.com/creationix/nvm/v0.33.2/install.sh | bash

NVM Depois da instalação，Reinicie o terminal e execute o seguinte comando para instalar Node.js :

nvm install stable

Siga este tutorial para instalar：Mac Instalar em Appium Detalhado

instalação QScan

usar NPM instalação Linus

Instalação global

npm install qscan [-g]

Depois que a instalação for concluída, você pode chamá-lo através da linha de comando，realizado qscan -v Visão QScan versão

Instale no projeto

npm install qscan [--save-dev]

Instalar como dependência do projeto，Pode ser introduzido por meio de módulos
