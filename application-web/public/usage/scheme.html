Plano de uso

Aqui estão alguns exemplos de cenários de uso, e você pode aprender sobre as idéias de uso do QScan aqui.

Ler o código em vários telefones

Digitalize o código para um único dispositivo

Cenário de uso: um único dispositivo executa código de varredura autônomo e, quando várias pessoas o chamam, o programa QScan é chamado várias vezes para formar uma fila.

Como usar: veja Começo rápido

Vários telefones leem seus códigos

Cenário de uso: vários dispositivos digitalizam códigos, cada dispositivo pode digitalizar seu próprio código especificamente para evitar confusão.

Uso: Necessita configurar vários arquivos de configuração ou vários objetos. Chamados separadamente sem afetar um ao outro

Construir um servidor de código de varredura

Configure um serviço de digitalização de código e introduza o QScan na forma de um módulo no serviço Node.

Depois que o serviço recebe a solicitação de rede, ele liga para o celular para fazer a leitura do código por meio da API.
